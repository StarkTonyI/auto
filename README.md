# TeamSync

TeamSync is a modern web application designed for team collaboration and productivity. It provides a sleek, responsive interface for managing tasks, tracking progress, and communicating in real-time.

### Key Features
- Real-time chat powered by WebSocket for instant team communication and collaboration.
- Advanced UI components including dialogs, modals, dropdowns, tooltips, toasts, and more for a polished user experience.
- Data visualization with interactive charts (Chart.js and Recharts) and progress indicators.
- Form handling and validation using React Hook Form.
- Date picking and calendar components for scheduling.
- Resizable panels and layout management for customizable dashboards.
- Theme support (light/dark mode) and animations for enhanced interactivity.
- Carousel/sliders for media and content display.

### Technologies Used
- Frontend Framework: React 18 with TypeScript for type-safe, component-based development.
- Build Tool: Vite for fast development server and optimized production builds.
- Styling: Tailwind CSS with plugins (animate, merge) and utilities like class-variance-authority; supplemented by styled-components.
- **UI Library**: Extensive use of Radix UI primitive components for accessible, unstyled building blocks, combined with custom styling.
- **Icons & Motion**: Lucide React icons and Framer Motion for animations.
- **State Management**: Redux Toolkit with React Redux.
- **Routing**: React Router DOM v7.
- **Forms & Inputs**: React Hook Form, Input OTP, and various Radix controls (select, checkbox, switch, etc.).
- **Data Fetching**: Axios for API requests; JWT Decode for authentication handling.
- **Other Libraries**: Day.js/Date-fns for date manipulation, Swiper for carousels, Sonner/Vaul for notifications and drawers, CMDK for command menus, Next Themes for theming, React Resizable Panels for layouts.
- **Development Tools**: ESLint for linting, PostCSS/Autoprefixer for CSS processing.
